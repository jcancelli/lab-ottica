## Build commands

| Command               | Description                  |
|-----------------------|------------------------------|
|`./build.sh test`      | Compile and run tests        |
|`./build.sh run`       | Compile and run main program |
|`./build.sh build-test`| Compile tests                |
|`./build.sh build`     | Compile main program         |
